Materials Studio 8.0 安装

发表于 2016-07-17   |   分类于 Materials Studio   |     |   阅读次数

在研一上学期入学之后差不多两个月（2015年11月），我在导师的要求下开始接触Materials Studio（MS）。由于我本科不是材料专业，量子力学相关的东西全都没有学过，不过好在对软件方面还算拿手，运用这个软件的过程也还算顺利。这次整理一下MS8.0的安装，虽然整个过程比较傻瓜，但是有的问题如果没有搜到正确的解决方法，会浪费很多时间。

介绍



Materials Studio作为一款可以全尺度模拟材料结构性能的软件，可以说做得非常的好用，从建模，设置参数，优化结构，到结果分析都非常的直观。说到建模，你会发现它自带的工具很充足，不管是手动建模，还是坐标输入，都可以满足要求，而且模型很好看，给人第一印象就很好，跟这款软件的界面一样。下面放几张图感受一下：

下载


推荐在小木虫MS8.0下载，下载下来的是一个种子文件，使用迅雷将其打开，会发现可以下载的有windows版本和linux版本，经测试都能用，我们选择win版本。
虽然这个软件是需要向创腾公司购买才能使用的，但是不作为商业用途和发表文章的话，下载下来用用应该是没有问题。据说购买的话要几十万，幸运的是我们学校买了。软件所需的许可一并在下载里面。

安装


下载的是.iso文件，需要我们电脑上有虚拟光驱软件，没有的话可以下载一个，推荐UltraISO软碟通，小巧方便。通过UltraISO加载Accelrys_Materials_Studio_8.0_Win.iso，然后运行setup.exe开始安装。
需要注意的是，安装目录的选择，可以随便在哪个盘，但是一定不要有中文！不然后面会出现问题。出现需要选择合适安装许可证的时候，我们选择最下面的For more options, start the License Administrator。
接下来是lisence的安装，使用记事本打开下载的sci.lic文件，将this_host改为自己的计算机名，也不能有中文，如果有中文，先修改自己的计算机名。改好之后，将文件放进C:\Program Files (x86)\Accelrys\LicensePack\Licenses\或者是C:\Program Files\Accelrys\LicensePack\Licenses\（取决于Arrelrys文件夹在哪个文件夹下面）。
接下来，最后一步，打开开始菜单，在所有程序>Accelrys>License>点击右键选择管理员方式打开Configure Materials Studio 8.0 Licensing。在License file > Install License点击Browse，选中上一步Licenses文件夹下的msi.lic文件，安装license，整个MS8.0的安装就完成了，各个计算板块都是能用的。
MS8.0的模块我用的比较多的主要是CASTEP和Dmol3，这两个都是基于量子力学进行第一性原理计算的工具。

Tips:

  • 我用Win10也装过，不过多少总会出点问题，用Win7不会出问题。
